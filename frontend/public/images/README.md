# Placeholder Images

- `hero-jacket-placeholder.jpg` - Hero slider image 1
- `hero-shirt-placeholder.jpg` - Hero slider image 2
- `hero-baggy-placeholder.jpg` - Hero slider image 3
- `product-placeholder.jpg` - Default product image

All images should be optimized for web use. Recommended dimensions:
